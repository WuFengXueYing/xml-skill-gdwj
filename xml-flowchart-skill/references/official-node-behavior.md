### 1.1.4. 清晰度检测

#### **1.1.4.1.** ***\*功能概述\****

开发的图像清晰度检测工具。该工具采用**拉普拉斯梯度方法**，通过计算图像经过拉普拉斯算子处理后的响应值来量化评估图像的清晰度。输出值越高，表示图像越清晰。

该方法的核心原理是：清晰图像包含更多边缘和细节（高频信息），其拉普拉斯算子的响应值也更高；而模糊图像边缘信息较少，响应值则较低。

#### 1.1.4.2. 输入参数 (ImageClarityInputParam)

| ***\*参数名\**** | ***\*类型\**** | ***\*说明\****             | ***\*必填\**** |
| ---------------- | -------------- | -------------------------- | -------------- |
| Image            | Mat            | 输入的源图像（OpenCV格式） | 是             |

#### 1.1.4.3. 输出参数 (ImageClarityOutputParam)

| ***\*参数名\**** | ***\*类型\**** | ***\*说明\****                                               |
| ---------------- | -------------- | ------------------------------------------------------------ |
| MeanValue        | double         | 图像的清晰度评价值。此值为拉普拉斯梯度响应的绝对值均值，并进行了2倍放大以增强区分度。***\*值越大，表示图像越清晰\****。 |

## 1.2. 逻辑组

### 1.2.1. 分支

#### 1.2.1.1. 功能概述

BranchStart是一个流程控制工具，用于根据条件表达式判断执行不同的流程分支。该工具会依次评估各分支的条件表达式，当某个条件满足时，流程将跳转到对应的分支继续执行。

#### 1.2.1.2. 输入参数

本工具不需要显式的输入参数，分支条件通过流程上下文中的ConditionBranchs集合进行配置。

#### 1.2.1.3. 输出行为

| ***\*行为\**** | ***\*说明\****                                  |
| -------------- | ----------------------------------------------- |
| 条件匹配       | 跳转到第一个满足条件的分支对应的Target节点      |
| 无匹配         | 跳转到默认的end节点（当前节点ID的"s"替换为"e"） |
